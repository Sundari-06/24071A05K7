# Library Management

A React application for managing library operations including user registration, login, book catalogue, issuing and returning books.

## Features

- User Registration
- User Login
- Book Catalogue
- Issue Book with Date
- Return Book with Date
- About Page

## Getting Started

1. Clone the repository
2. Run `npm install`
3. Run `npm run dev` to start the development server
4. Run `npm run build` to build for production

## Deployment

The app is set up for deployment to GitHub Pages via GitHub Actions.
